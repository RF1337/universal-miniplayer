Privacy Policy for Universal MiniPlayer

Last updated: 2026-06-24

Universal MiniPlayer is a browser extension that lets you pop the current tab out into a floating Picture-in-Picture window.

Data Collection

Universal MiniPlayer does not collect, store, or transmit any data. It has no analytics, no remote servers, and makes no network requests of any kind.

How Capture Works

When you click the extension's button, the extension captures the active tab's audio/video output locally, using Chrome's built-in tabCapture API, and plays that stream in a Picture-in-Picture window. This capture happens entirely within your browser. The captured stream is never saved, recorded, transmitted, or shared — it exists only as a live, temporary stream that is discarded when the Picture-in-Picture window is closed.

Permissions Used

  • activeTab — lets the extension act on the tab you're currently viewing, only when you click the extension's button.
  • tabCapture — lets the extension capture that tab's audio/video so it can be shown in Picture-in-Picture.
  • scripting — lets the extension inject the small script that starts and displays the Picture-in-Picture player on the active tab.

None of these permissions are used to read, log, or transmit the content of your tabs anywhere.
